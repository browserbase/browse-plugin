# Browserbase Functions

Deploy serverless browser automation to the Browserbase cloud.

This plugin provides a skill that guides Cursor through creating, testing, and deploying Browserbase Functions -- serverless browser automation that runs on schedules or webhooks.

## Prerequisites

- Browserbase account ([browserbase.com](https://browserbase.com))
- `BROWSERBASE_API_KEY` and `BROWSERBASE_PROJECT_ID` environment variables

## Usage

Ask Cursor to deploy browser automation:

- *"Create a Browserbase function that scrapes product prices daily"*
- *"Deploy a webhook that takes a screenshot of any URL"*
- *"Set up a cloud function for automated login testing"*
